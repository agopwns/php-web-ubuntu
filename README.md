# Peanut Community
웹 기반 커뮤니티 사이트입니다.   
  
멤버 기능과 기본적인 게시판 기능, 실시간 알림, 랜덤 채팅, 영상 스트리밍(방), 이더리움 후원 기능을 개발하였습니다.  
    
작품의 상세 작동 영상은 [여기](https://www.youtube.com/watch?v=XmwVZDh1gg4)에서 보실 수 있습니다.
